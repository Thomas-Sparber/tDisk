struct sorted_sector_index

A sorted_sector_index represents a physical sector sorted according to the access_count.

This struct is used for two purposes:

  • total_sorted: represents the physical sector sorted by all all physical sectors of th tDisk
  • device_assigned: is used to assign the sorted sectors to the sorted devices. This way one object can be used for both purposes
