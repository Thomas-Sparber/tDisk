struct device_performance

This struct represents performance indicators of a physical disk.

It contains the average and standard deviation values in processor cycles of read and write operations. Additionally, the residue of the averaging operations is stored to be able to use fixed point numbers instead of floating point.

e.g (1 << MEASUE_RECORDS_SHIFT) –> (1 << 16) = 65536
