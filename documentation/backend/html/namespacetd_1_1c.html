struct td::c::f_device_performance

Frontend version This struct is used for the "ADD_DISK" ioctl to add a specific device to a tDisk.

Frontend version This struct represents performance indicators of a physical disk. It contains the average and standard deviation values in processor cycles of read and write operations. Additionally, the residue of the averaging operations is stored to be able to use fixed point numbers instead of floating point.
